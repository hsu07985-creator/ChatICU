## 2. 用藥數據一致性 ✅

### Medication 資料結構
```typescript
interface Medication {
  id: string;
  patientId: string;
  name: string;          // 藥品名稱
  dose: string;          // 劑量
  route: string;         // 給藥途徑
  frequency: string;     // 頻率
  category: 'S' | 'A' | 'N' | 'other';  // 分類
  startDate: string;     // 開始日期
  status: 'active' | 'discontinued';    // 狀態
}
```

- **張三 (I-1)**:

- **李四 (I-2)**:
  - Propofol 50mg/hr IV infusion (S - 鎮靜)
  - Fentanyl 50mcg/hr IV infusion (A - 止痛)
  - Cisatracurium 2mg/hr IV infusion (N - 肌肉鬆弛)
  
- **王五 (I-3)**:
  - Dexmedetomidine 0.5mcg/kg/hr IV infusion (S - 鎮靜)
  - Morphine 2mg IV q6h prn (A - 止痛)

- **趙六 (I-4)**:
  - 無 S/A/N 用藥
