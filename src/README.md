# ChatICU - 智慧型加護病房照護系統

## 系統簡介

ChatICU 是一個專為加護病房設計的智慧型照護系統，整合了 AI 對話助手、病患資料管理、檢驗數據追蹤、用藥管理及藥事支援等功能。

## 核心功能

### 三種角色權限

1. **一般醫護**
   - 查看病患清單與詳細資料
   - 使用 AI 對話助手
   - 檢視檢驗數據與用藥記錄
   - 產出報告與交班資訊

2. **管理者**
   - 所有醫護功能
   - 上傳與管理文件（OCR 解析）
   - 管理向量資料庫
   - 查看稽核紀錄與系統健康
   - 帳號與權限管理
   - AI 模型設定

3. **藥師**
   - 藥物交互作用查詢
   - IV 相容性檢核
   - 劑量計算與建議
   - 用藥錯誤回報

## 測試帳號

- **一般醫護**: `nurse` / `nurse`
- **管理者**: `admin` / `admin`
- **藥師**: `pharmacist` / `pharmacist`

## 主要頁面

### 病人照護

- **總覽（儀表板）**: 插管病患數、異常警示、使用 S/A/N 藥物統計
- **病人清單**: 所有病患資料表格檢視
- **病人頁（Case Room）**: 包含 5 個分頁
  - 對話助手: AI 輔助臨床決策
  - 檢驗數據: 生化與血液檢驗結果
  - 用藥: 藥品清單與 S/A/N 標記
  - 病歷摘要: 診斷、重大事件、風險警示
  - 稽核: 操作記錄（僅管理者）

### 其他聊天

- **團隊聊天室**: 團隊溝通、公告釘選

### 資料管理（僅管理者）

- **文件與截圖上傳**: OCR 自動識別與欄位映射
- **向量資料庫**: RAG 知識庫管理
- **資料備份與還原**: 系統資料保護

### 紀錄（僅管理者）

- **稽核紀錄**: 操作足跡追蹤
- **系統健康與任務佇列**: 監控系統狀態
- **帳號與權限**: 使用者管理
- **模型與 AI 能力**: RAG 策略與 Prompt 設定

### 藥事支援中心（僅藥師）

- **交互作用查詢**: 藥物併用風險評估
- **相容性檢核**: IV 藥物配伍相容性
- **劑量計算與建議**: 個人化劑量調整
- **用藥錯誤回報**: 用藥安全監控

## 使用流程

### 一般醫護日常流程
```
登入 → 總覽 → 選擇病患 → 病人頁
        ├─ 對話助手諮詢
        ├─ 檢視檢驗數據
        ├─ 查看用藥記錄
        └─ 匯出報告
```

### 管理者資料整備
```
登入 → 資料管理 → 上傳檢驗單
        ↓
    OCR 解析 → 映射模板 → 覆核
        ↓
    指派到病患與時間點
```

### 藥師藥事查詢
```
登入 → 藥事支援中心
        ├─ 交互作用查詢
        ├─ 相容性檢核
        ├─ 劑量計算
        └─ 錯誤回報
```

## 資料結構

### 病患資料
- 基本資訊：姓名、床號、年齡、診斷
- 插管狀態與天數
- S/A/N 藥物使用標記
- 同意狀態與警示

### 檢驗數據
- 生化：Na、K、Cl、BUN、Cr、Glucose、ALT、AST、TBil
- 血液：WBC、Hb、Plt、Seg、Lymph
- 版本控制與校正記錄

### 用藥記錄
- 藥品名稱與類型（S/A/N）
- 開立日期、劑量、頻次、路徑
- 狀態（使用中/停用）
